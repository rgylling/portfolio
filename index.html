Project Highlights

Hoppie is a friendly application designed to find local breweries near you. This app allows you to search breweries near your location as well as provided a blood alcohol content calculator to make sure you and your friends make it home safely. This app was built with HTML, CSS, Javascript, jQuery, Foundations framework, page.js, handlebar.js, node.js, express.js, postgresql, jquery UI, Yelp API, Google maps API, AJAX, JSON, SMACSS, MVC, and CRUD. Click Here to visit the website!

La Brigada de Brasura is a website created for charity that was started by a Code Fellows Student. La Brigada de Brasura (the trash brigade) is a charity that enlists the help of local children to clean up the community. The charity was faced with the dilemma of having no way to accept donations via the internet. We decided to go ahead and bring La Brigade de Brasura into 2016 with a brand new website! The website was built using HTML, CSS, JavaScript, jQuery, Jekyll, and Bootstrap framework. Click Here to visit the website!
